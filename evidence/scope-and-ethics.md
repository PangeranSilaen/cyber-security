# Scope Dan Etika Pengujian

## Target Resmi

- `https://perpustakaan.itk.ac.id/`

## Dasar Kegiatan

Kegiatan ini dilakukan untuk UAS mata kuliah Keamanan Siber dalam bentuk project kelompok identifikasi kerentanan website. Aktivitas memiliki izin akademik, tetapi tetap dibatasi pada identifikasi, observasi, dan analisis non-destruktif.

## Aktivitas Yang Diperbolehkan

- Reconnaissance
- Footprinting
- Information gathering dari sumber publik
- Port scanning wajar dan konservatif
- Service enumeration
- Banner grabbing
- DNS enumeration
- Vulnerability scanning non-destruktif
- Identifikasi potensi misconfiguration berdasarkan bukti awal

## Aktivitas Yang Dilarang

- Eksploitasi aktif
- Payload/RCE/privilege escalation
- Brute force login atau credential stuffing
- DoS/DDoS atau stress testing
- Defacement
- Upload file berbahaya
- Pencurian, perubahan, atau penghapusan data
- SQL injection eksploit aktif

## Prinsip Pelaporan

Temuan ditulis sebagai potensi atau indikasi kecuali bukti non-destruktif cukup kuat. Evidence yang disimpan hanya metadata teknis non-sensitif seperti DNS record, header, service banner, TLS certificate metadata, dan output scanner aman.
